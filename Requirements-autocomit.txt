Issue 1: Проверка текущей директории на наличие репозитория Git
Задача:
    • Написать скрипт, который будет проверять, находится ли текущая директория в репозитории Git.
Issue 2: Добавление изменений в индекс
Задача:
    • Разработать команду для добавления всех изменений в индекс Git.
Issue 3: Проверка наличия изменений для коммита
Задача:
    • Написать проверку наличия изменений для фиксации в репозитории Git.
Issue 4: Фиксация изменений и создание коммита
Задача:
    • Написать команду для фиксации изменений в репозитории Git с автоматическим созданием коммита, содержащего текущую дату и время.
Issue 5: Вывод сообщений об успешном выполнении или отсутствии изменений
Задача:
    • Разработать вывод сообщений о результате выполнения скрипта: успешном фиксировании изменений или отсутствии изменений для коммита.
